# List

### Usage

```jsx
import { List } from 'drew-i-kit';

<List>
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
  <li>Four</li>
</List>
```

```jsx
import { List } from 'drew-i-kit';

<List type="disc">
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
  <li>Four</li>
</List>
<List type="decimal">
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
  <li>Four</li>
</List>
```

### Props

- `flush` (**Boolean**, optional)
- `type` (**string**, optional)
