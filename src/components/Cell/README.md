# Cell

## Usage

```jsx
import { Cell } from 'drew-i-kit';

<Cell>
  <p>Some content here...</p>
</Cell>
```

```jsx
import { Cell } from 'drew-i-kit';

<Cell center>
  <p>Some content here...</p>
</Cell>
<Cell size="s">
  <p>Some content here...</p>
</Cell>
<Cell size="l">
  <p>Some content here...</p>
</Cell>
```

## Props

- `center` (**Boolean**, optional)
- `minWidth` (**string**, optional)
- `size` (**string**, optional)
