# Image

## Usage

```jsx
import { Image } from 'drew-i-kit';

<Image src="https://source.unsplash.com/800x400" alt="Placeholder" />
```

```jsx
import { Image } from 'drew-i-kit';

<Image
  src="https://source.unsplash.com/800x400"
  alt="Placeholder"
  width="400"
  height="200" />
<Image
  src="https://source.unsplash.com/800x400"
  alt="Placeholder"
  width="400"
  height="200" />
```

## Props

- `alt` (**string**)
- `height` (**string**, optional)
- `onLoaded` (**Function**, optional)
- `src` (**string**)
- `width` (**string**, optional)
