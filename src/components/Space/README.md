# Space

## Usage

```jsx
import { Space } from 'drew-i-kit';

<Space bottom="base">
  <p>Some content here...</p>
</Space>
```

```jsx
import { Space } from 'drew-i-kit';

<Space left="base">
  <p>Some content here...</p>
</Space>
<Space right="s">
  <p>Some content here...</p>
</Space>
<Space top="xl">
  <p>Some content here...</p>
</Space>
```

## Props

- `bottom` (**string**, optional)
- `left` (**string**, optional)
- `right` (**string**, optional)
- `top` (**string**, optional)
